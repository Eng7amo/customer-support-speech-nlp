# customer-support-speech-nlp
End-to-end NLP and Speech Recognition project analyzing customer support calls using SpeechRecognition, spaCy, and sentiment analysis.


---

## Results Summary
- Successfully transcribed customer support audio calls
- Classified customer sentiment (positive / neutral / negative)
- Identified frequently mentioned entities
- Retrieved the most semantically similar customer call for a given query
